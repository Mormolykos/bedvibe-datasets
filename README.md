BedVibe Emotional Speech Datasets

Multilingual emotional speech datasets designed for training text-to-speech systems.

Features
• Multiple speakers
• Multiple languages
• Six emotional states (neutral, happy, angry, whisper, shouting, scared)
• Studio-recorded clean speech suitable for TTS training
